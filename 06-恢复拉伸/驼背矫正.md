## 什么是驼背（胸椎后凸增大）

正常脊柱有自然弯曲，胸椎本来就有一点后凸。但当后凸**角度过大**，就变成了我们说的"驼背"——外观表现为圆肩、含胸、头前伸。

### 驼背的三件套

```
① 圆肩（肩膀向前卷）
② 含胸（胸腔塌陷）
③ 头前伸（下巴前探）

这三个通常一起出现，互相加重。
```

### 自测方法

| 测试 | 做法 | 阳性表现 |
|------|------|----------|
| **靠墙测试** | 背靠墙站立，脚跟/臀/上背贴墙 | 后脑勺无法自然贴墙（需要用力仰头才能贴到） |
| **耳朵-肩膀对齐** | 侧面照镜子，看耳垂和肩峰的关系 | 耳垂明显在肩峰前方（头前伸） |
| **手背朝向** | 自然站立时观察手背朝向 | 手背朝前（说明肩膀内旋、圆肩） |
| **铅笔测试** | 双手各握一支铅笔自然下垂 | 铅笔尖指向内侧（圆肩），正常应朝前 |

> 如果靠墙测试后脑勺贴不到墙，基本确认有驼背倾向。

## 为什么会驼背

### 核心原因：前后肌肉失衡

| 位置 | 状态 | 涉及肌�� | 后果 |
|------|------|----------|------|
| **前面（胸/肩前）** | 过紧过短 | 胸大肌、胸小肌、肩内旋肌群 | 把肩膀向前拉 |
| **后面（上背）** | 过弱过长 | 菱形肌、中下斜方肌、深层颈屈肌 | 拉不住肩膀，背挺不起来 |
| **颈部前侧** | 过弱 | 深层颈屈肌 | 头前伸，下巴前探 |
| **颈部后侧** | 过紧 | 上斜方肌、枕下肌群 | 代偿性紧张，脖子酸痛 |

### 加重因素

| 因素 | 影响 |
|------|------|
| **久坐** | 每天坐 6+ 小时是最大元凶 |
| **看手机** | 低头角度 60° 时颈椎负重约 27kg |
| **没有背部训练** | 后背肌肉完全得不到刺激 |
| **单侧背包** | 一侧肩膀长期代偿 |
| **心理因素** | 焦虑/不自信 → 下意识蜷缩 → 变成习惯 |

## 拉伸动作（松开前面紧的）

### 1. 胸大肌门框拉伸

| 项目 | 说明 |
|------|------|
| **编号** | KB-S1 |
| **目标** | 胸大肌（上中下三束） |
| **动作** | 手臂撑在门框上，身体前倾，感受胸前拉伸 |
| **三个角度** | 手高于肩（拉下束）→ 手平肩（拉中束）→ 手低于肩（拉上束） |
| **时间** | 每个角度 每侧 30 秒 |
| **频���** | 每天 2 次（早晚各一次效果最好） |
| **感觉** | 胸前明显拉伸感，不应有肩前疼痛 |

### 2. 胸椎伸展（泡沫轴）

| 项目 | 说明 |
|------|------|
| **编号** | KB-S2 |
| **目标** | 恢复胸椎灵活度 |
| **动作** | 泡沫轴横放在上背（肩胛骨下方），双手抱头或交叉胸前，缓慢向后仰 |
| **分段做** | 泡沫轴放在上背��同位置（上/中/下），每个位置 8-10 次后仰 |
| **时间** | 共 2-3 分钟 |
| **注意** | **只在胸椎段做**（约肩胛骨到肋骨下缘），不要在腰部做 |

**记忆口诀**：轴放上背，手抱头，后仰开胸不弯腰。

### 3. 猫牛式（Cat-Cow）

| 项目 | 说明 |
|------|------|
| **编号** | KB-S3 |
| **目标** | 脊柱整体灵活度 |
| **动作** | 四点跪姿，交替做弓背（猫）和塌腰抬头（牛） |
| **次数** | 10 次 × 2 组 |
| **要点** | 动作缓慢，配合呼吸（弓背呼气、塌腰吸气） |
| **重点** | 牛式时特别关注胸椎的伸展，不要只动腰椎 |

### 4. 枕下肌群拉伸（颈部后侧）

| 项目 | 说明 |
|------|------|
| **编号** | KB-S4 |
| **目标** | 松解颈部后侧紧张（头前伸的代��） |
| **动作** | 坐直，双手交叉放在后脑勺，轻轻向下压（收下巴方向），保持 |
| **时间** | 20-30 秒 × 3 组 |
| **力度** | 轻柔，不要猛拉，颈椎很脆弱 |
| **感觉** | 后脑勺到颈根部有牵拉感 |

## 强化动作（练强后面弱的）

### 1. 收下巴训练（Chin Tuck）——最关键

| 项目 | 说明 |
|------|------|
| **编号** | KB-E1 |
| **目标肌肉** | 深层颈屈肌（矫正头前伸最关键的肌肉） |
| **动作** | 坐直或站直，保持视线平视，下巴向后收（做出"双下巴"的动作） |
| **次数** | 保持 5 秒 × 15 次 × 3 组 |
| **频率** | 每天多次，设闹钟提醒 |
| **要点** | 不是低头，是下巴水平向后缩，像乌龟缩头 |
| **进阶** | 空手 → 手指轻推下巴加阻力 → 弹力带加阻力 |

**记忆口诀**：不低头，做双下巴——像乌龟缩脖子。

### 2. 弹力带面拉（Face Pull）

| 项目 | 说明 |
|------|------|
| **编号** | KB-E2 |
| **目标肌肉** | 中下斜方肌、菱形肌、肩外旋肌群 |
| **动作** | 弹力带固定在门把手高度，双手拉向面部两侧，手肘高于肩膀，终点挤压肩胛骨 |
| **次数** | 15 次 × 3 组 |
| **要点** | 手肘朝后上方打开，终点停留 2 秒感受肩胛骨夹紧 |
| **注意** | 不要耸肩，颈部放松 |

### 3. W 字训练（贴墙版）

| 项目 | 说明 |
|------|------|
| **编号** | KB-E3 |
| **目标肌肉** | 下斜方肌、肩外旋肌群 |
| **动作** | 背靠墙站立，双臂做"W"形（手肘弯曲 90°）贴墙，沿墙上下滑动 |
| **次数** | 12 次 × 3 组 |
| **要点** | 手背、手肘、后脑勺都尽量贴墙 |
| **难度** | 如果后脑勺贴不到墙，说明驼背程度较重，先做到手肘贴墙即可 |

### 4. 俯卧 Y 字伸展

| 项目 | 说明 |
|------|------|
| **编号** | KB-E4 |
| **目标肌肉** | 下斜方肌（驼背人群最弱的肌肉之一） |
| **动作** | 俯卧，双臂向前上方举起呈 Y 字形，拇指朝上，抬离地面 |
| **次数** | 10 次 × 3 组（每次顶部停留 3 秒） |
| **注意** | 不要抬太高，感受肩胛骨下角发力即可 |
| **进阶** | 空手 → 0.5kg → 1kg |

### 5. 超人式（Superman）

| 项目 | 说明 |
|------|------|
| **编号** | KB-E5 |
| **目标** | 整体后链（背部伸肌群） |
| **动作** | 俯卧，同时抬起双手和双脚离地，保持 3-5 秒 |
| **次数** | 10 次 × 2 组 |
| **变式** | 对侧手脚交替抬起（更容易、更安全） |

## 姿势矫正习惯

| 场景 | 建议 |
|------|------|
| **坐姿** | 屏幕上缘与视线平齐；椅子有腰靠；每 30 分钟站起来 |
| **站姿** | 想象头顶有根绳子向上拉你；耳肩髋踝一条线 |
| **看手机** | 把手机举到视线高度，不要低头 |
| **睡姿** | 仰卧：枕头不要太高；侧卧：枕头填满颈肩间距 |
| **背包** | 双肩背包，调紧肩带让包贴背 |

### 30-30 法则（ADHD 友好版）

> **每坐 30 分钟，站起来做 30 秒收下巴 + 胸椎伸展。**
> 用手机定个重复闹钟，不需要记住——闹钟响了就做。

## 跑步相关影响

| 影响 | 说明 |
|------|------|
| **呼吸效率下降** | 胸腔打不开 → 肺活量下降 → 跑步喘得更快 |
| **摆臂受限** | 圆肩让手臂活动范围减小 → 浪费能量 |
| **颈肩疲劳** | 头前伸 → 颈部肌肉持续紧张 → 长距离后肩颈酸痛 |
| **与翼状肩胛互相加重** | 驼背 → 肩胛骨前倾 → 翼状肩胛 → 更驼背 |

## 常见误区

| 误区 | 真相 |
|------|------|
| "靠矫姿带就行了" | 矫姿带只是外力固定，不能增强肌肉；脱掉就回到原样 |
| "使劲挺胸就是好姿势" | 过度挺胸 = 腰椎过度前凸，同样有害；好姿势是自然中立位 |
| "驼背是骨头的问题" | 大多数年轻人的驼背是肌肉失衡导致的姿势性问题，可以通过训练改善 |
| "去健身房练胸肌" | 只练胸肌会让驼背更严重，应该多练背、少练胸（或至少平衡） |
| "看起来不严重就没事" | 轻度驼背不痛，但长期发展影响呼吸、消化、甚至情绪 |

## 什么时候该看医生

| 情况 | 建议 |
|------|------|
| 训练 12 周无明显改善 | 就诊康复科或运动医学科 |
| 背部持续疼痛（不只是酸） | 排除椎间盘等问题，就诊骨科 |
| 手臂麻木/无力 | 可能有神经压迫，就诊骨科/神经科 |
| 胸椎僵硬无法后仰 | 可能需要手法治疗，就诊康复科 |
| 青少年期快速加重 | 排除 Scheuermann 病（青少年结构性驼背） |
