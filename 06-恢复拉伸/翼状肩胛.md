# 翼状肩胛

> 肩胛骨像翅膀一样翘起来，不是因为你要飞，是因为"固定翅膀的肌肉"罢工了。

## 什么是翼状肩胛

肩胛骨的内侧缘或下角**明显突出于背部**，像翅膀一样翘起。正常人肩胛骨应该平贴在肋骨上。

### 自测方法

| 测试 | 做法 | 阳性表现 |
|------|------|----------|
| **墙面俯卧撑测试** | 面朝墙站立，双手撑墙做俯卧撑 | 推出时肩胛骨内侧明显凸起 |
| **镜子观察** | 侧身照镜子，自然放松站立 | 肩胛骨下角或内侧缘明显突出 |
| **手臂前举测试** | 双臂前举至水平 | 肩胛骨翘起，背部不平整 |
| **他人触摸** | 让别人从背后摸你的肩胛骨 | 能明显摸到凸起的骨头边缘 |

> 如果 2 项以上阳性，基本就是翼状肩胛了。

## 为什么会翼状肩胛

### 核心原因：前锯肌无力

**前锯肌**（Serratus Anterior）是一块"把肩胛骨贴在肋骨上"的关键肌肉，位于肋骨侧面。

```
正常：前锯肌有力 → 肩胛骨贴合肋骨 → 肩膀稳定
异常：前锯肌弱   → 肩胛骨失去固定 → 像翅膀一样翘起
```

### 完整原因链

| 原因 | 解释 | 自查方式 |
|------|------|----------|
| **前锯肌无力** | 最主要原因，肩胛骨失去"锚定" | 推墙/俯卧撑感觉肩膀不稳就是 |
| **胸小肌紧张** | 把肩胛骨向前拉，加重前倾 | 含胸、肩膀内扣 |
| **菱形肌/中下斜方肌弱** | 无法把肩胛骨向后拉回 | 驼背、肩膀前探 |
| **久坐含胸** | 长时间不良姿势让上述肌肉失衡 | 每天坐 6 小时以上 |
| **缺乏上肢训练** | 前锯肌日常很少被刺激到 | 不做俯卧撑、不举重物 |

## 拉伸动作（放松紧张的肌肉）

### 1. 胸小肌门框拉伸

| 项目 | 说明 |
|------|------|
| **编号** | WS-S1 |
| **目标肌肉** | 胸小肌、胸大肌 |
| **动作** | 手臂呈 90° 撑在门框上，身体前倾，感受胸前拉伸 |
| **角度变化** | 手高于肩 → 拉胸小肌下部；手平肩 → 拉胸大肌中部 |
| **时间** | 每侧 30 秒 × 3 组 |
| **频率** | 每天 1-2 次 |
| **感觉** | 胸前有明显牵拉感，**不应该有肩膀前侧的疼痛** |

### 2. 筋膜球胸小肌松解

| 项目 | 说明 |
|------|------|
| **编号** | WS-S2 |
| **目标肌肉** | 胸小肌深层筋膜 |
| **动作** | 面朝下趴在地上，筋膜球放在锁骨下方2-3指处（胸小肌位置），用体重压球缓慢滚动 |
| **时间** | 每侧 60-90 秒 |
| **力度** | 痛感 5-6/10（微痛但能忍受） |
| **注意** | 避开���骨和肩关节，只压肌肉部分 |

### 3. 胸椎伸展（泡沫轴）

| 项目 | 说明 |
|------|------|
| **编号** | WS-S3 |
| **目标** | 打开胸椎灵活度，减少圆背 |
| **动作** | 泡沫轴横放在上背（肩胛骨下方），双手抱头，缓慢向后仰 |
| **时间** | 10-15 次后仰 × 2 组 |
| **注意** | 只在胸椎段（上背）做，**不要在腰椎段做**（腰椎不适合大幅伸展） |

## 强化动作（激活无力的肌肉）

### 1. 前锯肌推举（最关键动作）

| 项目 | 说明 |
|------|------|
| **编号** | WS-E1 |
| **目标肌肉** | 前锯肌 |
| **动作** | 俯卧撑顶部位置（或撑墙），手臂伸直不动，只用肩胛骨做"推出"和"收回"的动作 |
| **要点** | 推出时感觉肩胛骨向两侧展开、贴紧肋骨；收回时感觉肩胛骨靠拢 |
| **次数** | 15 次 × 3 组 |
| **频率** | 每天或隔天 |
| **进阶** | 墙面 → 桌面 → 地面 → 负重 |

**记忆口诀**：手不弯，只推肩——像猫弓背一样把肩胛骨推出去。

### 2. 弹力带肩胛后缩

| 项目 | 说明 |
|------|------|
| **编号** | WS-E2 |
| **目标肌肉** | 菱形肌、中斜方肌 |
| **动作** | 双手持弹力带与肩同宽，手臂伸直向两侧拉开，挤压肩胛骨靠拢 |
| **次数** | 15 次 × 3 组 |
| **呼吸** | 拉开时呼气，回收时吸气 |
| **注意** | 不要耸肩，肩膀始终下沉 |

### 3. Y-T-W 训练

| 项目 | 说明 |
|------|------|
| **编号** | WS-E3 |
| **目标肌肉** | 下斜方肌(Y)、中斜方肌(T)、肩外旋肌群(W) |
| **动作** | 俯卧在地面或趴在斜板上，双臂分别做 Y（前上方举起）、T（侧方平举）、W（手肘弯曲外旋）三个姿势 |
| **次数** | 每个字母 10 次 × 2 组 |
| **注意** | 不用拿重物，空手做到位比拿重量更重要 |
| **进阶** | 空手 → 0.5kg 哑铃 → 1kg 哑铃 |

### 4. 高位平板支撑 + 前锯肌激活

| 项目 | 说明 |
|------|------|
| **编号** | WS-E4 |
| **目标肌肉** | 前锯肌 + 核心 |
| **动作** | 俯卧撑顶部位置保持，每 5 秒做一次肩胛骨推出（前锯肌推举） |
| **时间** | 30 秒 × 3 组 |
| **难度** | 中等，需要核心和前锯肌同时工作 |

## 跑步相关影响

| 影响 | 说明 |
|------|------|
| **摆臂效率下降** | 肩胛骨不稳定 → 手臂摆动浪费能量 |
| **上半���紧张** | 肩颈代偿 → 跑久了肩膀酸痛 |
| **呼吸受限** | 胸廓打不开 → 深呼吸困难 |
| **与驼背互相加重** | 翼状肩胛 ⇔ 驼背形成恶性循环 |

## 常见误区

| 误区 | 真相 |
|------|------|
| "多练俯卧撑就好了" | 普通俯卧撑主要练胸肌，前锯肌激活不够；需要专门的前锯肌推举 |
| "戴矫姿带能矫正" | 矫姿带只是被动固定，不能让肌肉变强，脱了就回去 |
| "只要拉伸就行" | 拉伸只解决"紧"的问题，还必须强化"弱"的肌肉，否则松了又紧 |
| "几天就能好" | 肌力建立需要 6-12 周，急不来 |
| "看起来没事就没事" | 翼状肩胛早期不痛，但长期影响肩关节健康 |

## 什么时候该看医生

| 情况 | 建议 |
|------|------|
| 拉伸和训练 8 周无改善 | 就诊运动医学科或康复科 |
| 伴随肩膀疼痛、手臂麻木 | 可能有神经问题（长胸神经损伤），立即就诊 |
| 单侧突然出现 | 排除神经损伤，就诊骨科/神经科 |
| 伴随肩关节弹响/卡顿 | 可能有肩关节病变，就诊骨科 |
