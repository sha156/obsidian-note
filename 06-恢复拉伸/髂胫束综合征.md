## 什么是髂胫束综合征（ITBS）

**髂胫束**（Iliotibial Band，简称 ITB）是一条从髋部外侧延伸到膝盖外侧的**厚实筋膜带**，不是肌肉，而是一条"绳子"。

当它反复在膝盖外侧骨突上摩擦，就会发炎疼痛——这就是**髂胫束综合征**，跑者膝外侧疼痛的头号原因。

---

## 自测方法

| 测试 | 做法 | 阳性表现 |
|------|------|----------|
| **按压测试** | 用手指按压膝盖外侧骨突处（股骨外上髁） | 明确的压痛点 |
| **Noble 压缩测试** | 按住膝外侧痛点，缓慢伸直膝盖 | 在膝盖弯曲约 30° 时触发疼痛 |
| **Ober 测试** | 侧卧，上方腿向后伸直放松，看能否自然下落 | 腿无法自然下落（髂胫束紧） |
| **跑步症状** | 跑步 2-3 公里后膝外侧开始痛，下坡加重 | 典型 ITBS 发作模式 |

> **关键特征**：疼痛在跑步固定距离后出现，停下来就缓解，再跑又痛。

## 为什么会得 ITBS

### 核心原因：臀肌无力

```
臀中肌弱 → 跑步时骨盆侧倾 → 髂胫束被反复过度拉扯
    ↓
髂胫束在膝盖外侧反复摩擦 → 发炎 → 疼痛
```

### 完整原因表

| 原因 | 解释 | 自查方式 |
|------|------|----------|
| **臀中肌无力** | 最主要原因，骨盆稳定性不足 | 单腿站立骨盆下沉（Trendelenburg 征） |
| **髂胫束紧张** | 长期紧张增加摩擦 | Ober 测试阳性 |
| **跑量突然增加** | 组织来不及适应 | 最近加量 >10%/周 |
| **下坡跑过多** | 下坡时膝外侧受力最大 | 越野跑者高发 |
| **O 型腿/足外翻** | 生物力学不良，增加外侧压力 | 鞋底外侧磨损严重 |
| **跑鞋磨损严重** | 缓震和支撑失效 | 跑鞋超过 800 公里未换 |
| **跑步姿势问题** | 过度跨步、膝盖内扣 | 跑步时膝盖向内"X"形 |

## 急性期处理（正在痛的时候）

| 阶段 | 做法 | 说明 |
|------|------|------|
| **立即** | 停止跑步 | 痛着跑只会更严重 |
| **前 48 小时** | 冰敷膝外侧 | 每次 15 分钟，每天 3-4 次 |
| **前 1-2 周** | 避免加重活动 | 不跑步、不下楼梯（可以游泳、骑车） |
| **同时开始** | 泡沫轴松解 + 臀肌强化 | 疼痛不剧烈就可以做（见下方） |

> **不建议**完全卧床休息——停止跑步但保持活动，越早开始臀肌训练恢复越快。

## 松解动作（放松髂胫束及周围肌肉）

### 1. 泡沫轴大腿外侧松解

| 项目 | 说明 |
|------|------|
| **编号** | IT-S1 |
| **目标** | 阔筋膜张肌（TFL）+ 股外侧肌 |
| **动作** | 侧卧在泡沫轴上，从髋部外侧滚到膝盖上方（**不滚膝盖**） |
| **时间** | 每侧 60-90 秒 |
| **重点区域** | 大腿上 1/3（TFL 区域）最容易紧，多停留 |
| **力度** | 痛感 5-7/10，不要硬忍剧痛 |
| **频率** | 每天或跑后必做 |

**重要提示**：直接滚髂胫束本身其实效果有限（它是筋膜不是肌肉），重点滚它上方的 **TFL**（阔筋膜张肌）和旁边的**股外侧肌**。

### 2. 泡沫轴臀部松解

| 项目 | 说明 |
|------|------|
| **编号** | IT-S2 |
| **目标** | 臀大肌、臀中肌 |
| **动作** | 坐在泡沫轴上，交叉一条腿（脚踝放在对侧膝盖上），侧向滚动臀部 |
| **时间** | 每侧 60 秒 |
| **作用** | 松解臀肌紧张，间接减少髂胫束张力 |

### 3. 站立 IT Band 拉伸

| 项目 | 说明 |
|------|------|
| **编号** | IT-S3 |
| **动作** | 站立交叉双腿（痛侧腿在后），身体向对侧侧弯，感受大腿外侧拉伸 |
| **时间** | 每侧 30 秒 × 3 组 |
| **加强版** | 靠墙做，用墙壁支撑身体侧弯得更深 |
| **注意** | 髂胫束是很难"拉松"的，这个拉伸主要作用是 TFL 和臀肌 |

### 4. 鸽子式拉伸

| 项目 | 说明 |
|------|------|
| **编号** | IT-S4 |
| **目标** | 臀深层肌群（梨状肌等） |
| **动作** | 一条腿弯曲放在身前，另一条腿向后伸直，身体前倾 |
| **时间** | 每侧 30-60 秒 |
| **作用** | 松解臀部深层紧张，减少对髂胫束的拉力 |

## 强化动作（解决根本原因）

### 1. 蚌式开合（Clamshell）——最关键

| 项目 | 说明 |
|------|------|
| **编号** | IT-E1 |
| **目标肌肉** | 臀中肌 |
| **动作** | 侧卧，双膝弯曲，脚并拢，上方膝盖向上打开再合上（像蚌壳一样） |
| **次数** | 20 次 × 3 组 每侧 |
| **进阶** | 空手 → 弹力带套在膝盖上方 → 更强弹力带 |
| **要点** | 骨盆不要向后翻转，保持身体侧面朝上 |

**记忆口诀**：脚不动，膝打开——像蚌壳开口。

### 2. 侧卧抬腿

| 项目 | 说明 |
|------|------|
| **编号** | IT-E2 |
| **目标肌肉** | 臀中肌、阔筋膜张肌 |
| **动作** | 侧卧伸直双腿，上方腿微微向后（脚尖朝下），向上抬起 30-45° |
| **次数** | 15 次 × 3 组 每侧 |
| **要点** | 脚尖朝下（避免髂胫束代偿），抬腿时臀部发力而不是腰 |

### 3. 单腿臀桥

| 项目 | 说明 |
|------|------|
| **编号** | IT-E3 |
| **目标肌肉** | 臀大肌、臀中肌 |
| **动作** | 仰卧，一条腿弯曲踩地，另一条腿伸直抬起，用踩地那侧臀部发力挺髋 |
| **次数** | 12 次 × 3 组 每侧 |
| **进阶** | 双腿臀桥 → 单腿臀桥 → 脚踩高处单腿臀桥 |
| **注意** | 挺髋时骨盆不要歪斜（痛侧下沉说明太弱了） |

### 4. 侧步走（弹力带）

| 项目 | 说明 |
|------|------|
| **编号** | IT-E4 |
| **目标肌肉** | 臀中肌功能性激活 |
| **动作** | 弹力带套在脚踝或膝盖上方，半蹲姿势侧向走 |
| **次数** | 每侧 15 步 × 3 组 |
| **频率** | 跑前热身必做 |
| **作用** | 模拟跑步时臀中肌的稳定功能 |

## 跑步调整

| 调整项 | 建议 |
|--------|------|
| **跑量** | 减到无痛量的 50%，每周增加不超过 10% |
| **地形** | 急性期避免下坡和弯道跑 |
| **路面** | 优先选平坦软路面（塑胶跑道、土路） |
| **步频** | 提高步频（180 步/分），减小步幅，降低膝盖冲击 |
| **跑前** | 5 分钟弹力带臀肌激活（蚌式 + 侧步走） |
| **跑后** | 泡沫轴松解 + 冰敷（如果还有不适） |
| **跑鞋** | 检查跑鞋是否该换，考虑足弓支撑鞋垫 |

## 恢复时间线

| 阶段 | 时间 | 做什么 |
|------|------|--------|
| **急性期** | 0-2 周 | 停跑、冰敷、开始松解和臀肌训练 |
| **恢复期** | 2-6 周 | 逐渐恢复低强度跑步 + 持续臀肌强化 |
| **重建期** | 6-12 周 | 恢复正常跑量，强化训练不能停 |
| **预防期** | 长期 | 跑前臀肌激活成为固定习惯 |

## 常见误区

| 误区 | 真相 |
|------|------|
| "使劲滚髂胫束就好了" | ITB 是筋膜不是肌肉，直接碾压效果有限且疼；重点松解 TFL 和臀肌 |
| "膝盖痛就是膝盖的问题" | ITBS 根源在臀部，只治膝盖永远好不了 |
| "忍着跑跑就好了" | 越跑越严重，可能发展成慢性炎症 |
| "休息就能好" | 不强化臀肌，一恢复跑步就会复发 |
| "去做个推拿松一松" | 推拿只能暂时缓解，不解决臀肌无力的根本问题 |
| "贴个膏药就行" | 外用药只能消炎止痛，不能改变生物力学 |

---

## 什么时候该看医生

| 情况 | 建议 |
|------|------|
| 休息 2 周 + 训练 4 周无改善 | 就诊运动医学科 |
| 走路也痛 | 可能不只是 ITBS，需要排除半月板等问题 |
| 膝关节肿胀/积液 | 可能有关节内问题，就诊骨科 |
| 疼痛伴弹响/卡锁 | 可能有其他结构损伤 |

> **推荐科室**：运动医学科 > 骨科 > 康复科。找运动医学科最对口。
