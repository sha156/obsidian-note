# 耗电触发器

> 了解什么在榨干我，是保护能量的第一步。

---

## 已知的耗电来源

### 🔴 高耗电（接触后迅速耗尽）

| 触发器 | 描述 |
|--------|------|
| **人多的环境** | 噪音、被注视、无法控制刺激输入 |
| **等待 / 排队** | 无法控制时间节奏，大脑空转又无法专注 |
| **重复机械的事** | 没有新刺激，强迫自己执行令大脑疲惫 |
| **大量社交 / 需要说很多话** | 需要持续管理输出，非常消耗 |

### 🟡 中耗电（累积后才明显）

- 感觉很委屈但说不清楚
- 努力了但和现实不匹配（付出没有回报）
- 差一点成功但失败了
- 因为 ADHD 做不好某件事，开始自我批评

---

## 情绪崩溃的触发链

以下模式容易导致情绪失控或崩溃：

```
觉得委屈 → 现实不匹配期望 → 快要成功但失败 → 自责"因为ADHD"
→ 情绪崩溃
```

**识别信号**：当开始自我批评、觉得"我就是不行"时，要意识到这是触发链在运行，不是真相。

---

## 感觉过载是什么样的

- 觉得很吵、很烦躁
- 无法正常处理信息
- 应对方式：带上耳机，制造安静

---

## 拖延的特殊耗电模式

**拖延本身也是一种耗电**：
不重要的事（如洗澡、做作业）→ 刷手机代替 → 感到自责自暴自弃 → 更难启动 → 继续拖延

这是一个闭环，越拖越耗电。打破它的方式见 [[03-Bug与补丁/有效策略]]。
