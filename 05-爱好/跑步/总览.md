# 跑步 总览

> 用双脚丈量世界，用心率测量自己。

---

## 基本概念

**跑步**：一种有氧运动，通过持续移动双腿使身体前进，是人类最古老、门槛最低的运动形式之一。核心价值在于提升心肺功能、释放压力、建立自我掌控感。

---

## 入门必须知道的10件事

1. **慢到你觉得"怎么这么慢"才是对的速度** — 新手最大误区是跑太快，能轻松说话的速度就是入门正确配速
2. **跑鞋是唯一不能省的装备** — 根据足弓类型（扁平/正常/高弓）选鞋，不按品牌贵贱选
3. **跑前动态热身，跑后静态拉伸** — 跑前不要压腿（降低爆发力），而是做抬腿、转髋等动态动作
4. **每周跑量增幅不超过10%** — 这是防伤黄金法则，软骨和肌腱需要时间适应
5. **心率比配速更重要** — 新手看心率，不要只盯着速度数字
6. **跑走结合是正确的入门方式** — "跑1分钟走2分钟"反复循环，比硬撑跑完5公里健康
7. **步频比步幅更关键** — 步频170-180步/分钟是理想区间，小步快跑比大步慢跑更护膝
8. **跑鞋有寿命** — 跑满600-800公里就需要换，不然缓震失效伤膝
9. **休息是训练的一部分** — 每周至少留1-2天完全休息，肌肉在休息时才生长
10. **找到跑团/跑友** — 一个人跑容易放弃，社群的力量是最强的坚持动力

## 术语表

| 术语 | 解释 |
|------|------|
| [[配速]] | 跑完每公里所需时间，如"配速6分"=1公里跑6分钟 |
| [[心率区间]] | 将心率分成5个区，区间2-3适合日常跑步训练 |
| [[步频]] | 每分钟脚落地次数，理想值170-180步/分 |
| [[步幅]] | 两脚落地间距，过大会增加膝盖冲击 |
| [[PB]] | Personal Best，个人最好成绩 |
| [[LSD]] | Long Slow Distance，长距离慢跑，提升基础耐力 |
| [[间歇跑]] | 快跑与慢跑交替，提升最大摄氧量和速度 |
| [[节奏跑]] | 介于轻松跑和间歇跑之间的强度，提升乳酸阈值 |
| [[最大摄氧量]] | VO2max，衡量有氧能力的黄金指标 |
| [[跑者膝]] | 髌骨软化或IT Band综合征，跑步最常见伤病 |
| [[撞墙]] | 马拉松约30km处糖原耗尽引发的极度疲劳感 |
| [[BQ]] | Boston Qualify，达到波士顿马拉松报名标准，顶级荣誉 |
| [[跑团]] | 定期一起跑步的社群组织 |
| [[能量胶]] | 比赛/长跑中补充糖分的胶状食品 |
| [[碳板鞋]] | 内置碳纤维板的竞速跑鞋，提升推进效率 |

## 圈子里常聊的话题

- **配速与PB** — "你5K最快多少？""这次马拉松目标配速多少？"
- **装备测评** — 跑鞋哪双好、手表用什么、碳板鞋值不值
- **赛事体验** — 上马/北马/无锡马拉松氛围、补给、路线难度
- **伤病与恢复** — 膝盖怎么样了、足底筋膜炎怎么处理
- **跑团活动** — 周末约跑、跑后吃什么、谁新加入了
- **训练计划** — 备赛期怎么安排、心率区间、力量训练加多少
- **跑步与生活** — 早跑还是晚跑、如何平衡工作和训练
- **补给策略** — 能量胶什么时候吃、电解质怎么补

1. **波士顿马拉松** — 世界最古老马拉松，有资格门槛（BQ），跑者圣地

## 新手常见误解

| 误解 | 真相 |
|------|------|
| **"跑步会伤膝盖"** | 姿势不当+跑量过猛才伤膝，循序渐进的跑步反而能强化关节 |
| **"跑前要拉伸（压腿）"** | 跑前做动态热身，静态拉伸留到跑后 |
| **"坚持就是胜利，忍着跑"** | 疼痛是警报，轻微的跑开了不等于没事，持续疼痛立即停跑 |
| **"越贵的鞋越好"** | 适合自己足型才是好鞋，错误类型的鞋贵也白搭 |
| **"每天跑才有效"** | 肌肉需要48小时恢复，刚开始每周3次足够 |
| **"跑慢了没用"** | 80%的跑步应该是慢跑，慢跑才是建立有氧基础的核心 |
| **"马拉松就是42.195公里"** | 更常见的是5K、10K、半马（21.0975km）等距离 |
