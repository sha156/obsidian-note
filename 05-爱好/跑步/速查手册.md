# 跑步 速查手册

> 见跑者朋友前5分钟看这个

---

## 最重要的3个概念

- **配速** = 跑1公里要多少分钟（"配速6"= 1公里6分钟）
- **PB** = 自己某距离跑过的最好成绩
- **心率区间** = 用心率控制强度，新手目标是能轻松说话的强度

---

## 距离速查

| 名称 | 距离 | 说明 |
|------|------|------|
| 5K | 5公里 | 新手目标距离 |
| 10K | 10公里 | 进阶目标 |
| 半马 | 21.0975公里 | Half Marathon |
| 全马 | 42.195公里 | 马拉松正式距离 |
| 100K/越野 | 不定 | 超马/越野领域 |

## 中国最热赛事（难中签排序）

1. **上海马拉松** — 最难中签，国际水准
2. **无锡马拉松** — 跑PB首选，快速赛道
3. **北京马拉松** — 历史最悠久
4. **厦门马拉松** — 冬季赛，风景好
5. **广州马拉松** — 南方跑者热门

---

## 主流跑鞋品牌一句话定位

- **HOKA** — 厚底缓震，最火，适合长距离
- **Brooks** — 专业跑步品牌，跑者口碑最佳
- **On昂跑** — 瑞士设计，好看+舒适
- **Nike Vaporfly/Alphafly** — 碳板竞速，成绩党首选
- **Garmin手表** — 跑者数据神器

## 跑步圈黑话速查

| 说法 | 意思 |
|------|------|
| 配速5分 | 1公里跑5分钟 |
| 跑进330/400 | 全马跑进3:30/4:00 |
| 撞墙了 | 马拉松30K后极度疲劳 |
| 破三 | 马拉松跑进3小时，高手标志 |
| 大满贯 | 六大世界马拉松全部完赛 |
| 跑力值 | Garmin的综合跑步能力评分 |
| 骨折价 | 促销价（跑鞋购买话题） |

---

## 当下趋势（2025）

- **碳板鞋**普及，越来越多普通跑者用竞速鞋
- **HOKA和On昂跑**强势增长，挑战Nike/Adidas
- **跑步社交化**：跑后咖啡、跑后聚餐是常态
- **马拉松"减量提质"**：精品赛事更难报名，小赛事在减少
- **文体旅融合**：去外地跑马拉松顺便旅游已成主流
