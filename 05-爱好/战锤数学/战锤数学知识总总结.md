## 1.2 骰子期望值

$$E(\text{Dn}) = \frac{n+1}{2}$$

| 骰子 | 期望值 | 常见用途 |
|------|-------|---------|
| D3 | 2.0 | 部分伤害值 |

| D6+6 | 9.5 | 暴虐巨兽主炮、征服者主炮 |
| 3D3 | 6.0 | 等离子焚烧器攻击次数 |

## 1.3 三道门伤害公式

$$E_{\text{dmg}} = A \times P_{\text{hit}} \times P_{\text{wound}} \times P_{\text{save fail}} \times D \times P_{\text{fnp fail}}$$

| 符号 | 含义 | 计算方式 |
|------|------|---------|
| $A$ | 攻击次数 | 武器档案直接读取或掷骰期望值 |
| $P_{\text{hit}}$ | 命中概率 | 命中值对应的 D6 成功率 |
| $P_{\text{wound}}$ | 伤害概率 | 由 S vs T 关系查下表 |
| $P_{\text{save fail}}$ | 护甲失败率 | $1 - P_{\text{save}}$，考虑 AP 修正 |
| $D$ | 伤害值 | 固定值或骰子期望值 |
| $P_{\text{fnp fail}}$ | 无痛失败率 | $1 - P_{\text{fnp}}$（无无痛则为1） |

**核心性质：全部是乘法（交换律），计算顺序不影响结果。**

> 数据来源：Part4 视频字幕 — "How to Mathhammer" 教程

## 1.6 重投（Reroll）价值公式

$$P_{\text{reroll}} = P_{\text{原始}} \times \frac{6 + n}{6}$$

其中 $n$ = 可重投的骰面数量。

## 1.7 特殊规则等效值速查

### 致命命中（Lethal Hits）— 跳过伤害门

$$\text{暴击等效值} = \frac{1}{P_{\text{wound}}} = \frac{6}{\text{伤害门成功骰面数}}$$

| 伤害门 | 通过率 | 暴击等效值 | 价值评级 |
|--------|-------|-----------|---------|
| 2+ | 5/6 | 1.2 | 几乎无用 |
| 3+ | 4/6 | 1.5 | 小幅 |
| 4+ | 3/6 | 2.0 | 中等 |
| 5+ | 2/6 | **3.0** | 高价值 |
| 6+ | 1/6 | **6.0** | 极高 |

### 毁灭性伤害（Devastating Wounds）— 跳过护甲门

$$\text{暴击等效值} = \frac{1}{P_{\text{save fail}}} = \frac{6}{\text{护甲失败骰面数}}$$

| 对手保护(含AP) | 失败率 | 暴击等效值 | 价值评级 |
|---------------|-------|-----------|---------|
| 无保护 | 6/6 | 1.0 | 无用 |
| 4+保护 | 3/6 | 2.0 | 中等 |
| 2+保护 | 1/6 | **6.0** | 极高 |

### 持续命中（Sustained Hits X）

直接在暴击骰面加 $X$。示例：命中3+/持续2 → 骰面6值 = 1+2 = 3 → 总期望 6/6 = 1.0。

### 致命 vs 持续 选择总表

| 伤害门 | 致命等效 | 持续1等效 | **选择** |
|--------|---------|----------|---------|
| 2+(5/6) | 1.2 | 2 | **持续** |
| 3+(4/6) | 1.5 | 2 | **持续** |
| 4+(3/6) | 2.0 | 2 | 持平 |
| 5+(2/6) | 3.0 | 2 | **致命** |
| 6+(1/6) | 6.0 | 2 | **致命** |

> **口诀：以上打下选持续，以下犯上选致命。**

> 数据来源：Part4 视频字幕 — "How to Mathhammer" + Part3 — Vindicator 黑暗契约分析

## 1.8 伤害值修正顺序 SPEMDAS

$$\text{Set} \to \text{PEMDAS}$$

**关键：半伤在加法之前** → 融毁(melta)+2 不受半伤影响。

# 6. 实战思考与计算指南

## 6.1 实战中应该怎么思考

### 思考流程图

```
第1步：读取场景信息
├── 我的单位：攻击次数、S、AP、D、特殊规则
├── 目标单位：T、保护、无敌、W、无痛、特殊规则
└── 环境：掩体、Buff可用性、CP剩余

第2步：快速判断"门"的难度
├── 命中门：命中值 → 查概率表
├── 伤害门：S vs T → 查阈值表
├── 护甲门：AP vs 保护/无敌 → 查失败率表
└── 找出"最卡"的一道门 = 优先提升目标

第3步：决定Buff分配
├── 最卡门是伤害门(5+/6+) → 致命命中 / 伤+1
├── 最卡门是护甲门(2+/1+) → 毁灭性伤害 / +AP
├── 门都还行(3+/4+) → 持续命中 / 重投
└── 不要在已经容易的门上浪费Buff

第4步：快速估算期望值
├── 心算：攻击 × 命中分数 × 伤害分数 × 护甲失败分数 × D
├── 对比目标W → 判断���否击杀
└── 低于75%击杀概率 → 考虑是否值得投入

第5步：做出决策
├── 能杀(>75%) → 执行，选稳定Buff
├── 可能杀(50-75%) → 接受风险，选爆发Buff
├── 杀不了(<50%) → 换目标或分散火力
└── 绝望(<25%) → 不投资此攻击
```

## 6.2 快速计算流程（步步拆解）

### 步骤1 — 收集数据

| 需要的信息 | 从哪里找 | 示例 |
|-----------|---------|------|
| 攻击次数(A) | 武器档案 | 10发(Hellblasters×10) |
| 命中值(BS/WS) | 单位档案 | 命中3+ |
| 力量(S) | 武器档案 | S8 |
| 韧性(T) | 目标档案 | T5(终结者) |
| AP | 武器档案 | AP-3 |
| 目标保护 | 目标档案 | 2+保护/4++无敌 |
| 伤害值(D) | 武器档案 | D2 |
| 目标血量(W) | 目标档案 | 3W(终结者) |
| 特殊规则 | 双方档案 | 过热/致命命中/无痛等 |

### 步骤2 — 逐门计算

| 门 | 查什么 | 得到什么 | 示例 |
|----|--------|---------|------|
| 命中 | 命中值→概率表 | $P_{hit}$ | 命中3+ → 4/6 |
| 伤害 | S vs T→阈值表 | $P_{wound}$ | S8 vs T5 → 3+ → 4/6 |
| 护甲 | AP vs 保护→修正表 | $P_{save fail}$ | AP-3 vs 2+ → 5+ → 失败4/6；但4++无敌 → 失败3/6 |
| 无痛 | 有无FNP | $P_{fnp fail}$ | 无 → 1.0 |

### 步骤3 — 乘法合计

$$E = 10 \times \frac{4}{6} \times \frac{4}{6} \times \frac{3}{6} \times 2 = 10 \times 0.667 \times 0.667 \times 0.5 \times 2 = \mathbf{4.44} \text{ 伤}$$

$$\text{击杀数} = \frac{4.44}{3W} = \mathbf{1.48} \text{ 个终结者}$$

### 步骤4 — 加入 Buff 修正

| Buff | 修正方式 | 新值 |
|------|---------|------|
| 旧誓言(重投命中) | $\times \frac{6+2}{6}$ | 4.44 × 8/6 = **5.92** |
| 新誓言(重投+伤+1) | 伤门变2+ → $\times \frac{5}{4}$ | 5.92 × 5/4 = **7.40** |
| +中尉(致命命中) | 暴击等效 +1.5 | ~8.0 |

## 6.5 不同场景下的决策模板

### 模板A — 射击阶段

```
1. 选择目标（誓言时刻目标优先）
2. 查 S vs T → 确定伤害门难度
3. 查 AP vs 保护 → 确定护甲门难度（注意掩体-1AP）
4. 找最卡的门 → 分配对应Buff
5. 快速心算期望值 → 与目标W对比
6. 期望 > W×1.3 → 大概率击杀，执行
7. 期望 < W×0.5 → 换目标
```

### 模板B — 近战阶段

```
1. 确认冲锋加成（+1攻击等）
2. 查 S vs T → 冲锋后S值可能变化
3. 评估对手反击能力 → 我方单位能否存活
4. Buff选择：
   - 目标T低 → 持续命中（量大取胜）
   - 目标T高 → 致命命中（跳过伤害门）
   - 目标2+甲 → 毁灭性伤害（跳过护甲门）
5. 心算 → 判断能否一回合击杀
6. 不能一回合杀 → 考虑是否值得冲锋（被反击风险）
```

### 模板C — 反坦克

```
1. 目标档案：T / 保护 / 无敌 / W / 特殊(-1D/半伤)
2. 有无无敌：
   - 无无敌 → Eradicators等少发高伤武器可用
   - 有无敌 → 优先大量攻击次数的武器
3. 有无半伤：
   - 有 → 需要高基础D值（D3+以上）
   - 避免D1武器
4. 优先使用角斗士枪骑兵（SM最优反坦克）
5. 多单位集火 > 分散射击
6. 预留1CP用于关键武器重投
```

### 模板D — 持久战（守点/消耗）

```
1. 我方占点单位耐久评估：
   - 普通步兵 → 对手小武器即可清除
   - 重截击者(目标上) → 对D1武器等效2+保护
   - 护刃/终结者 → 有无敌，抗高AP
2. 对手需要多少资源清除我方：
   - 如果对手需要用大火力清小目标 = 赚
3. 不要过度投入攻击：
   - 留单位占点 > 全部前压冲锋
4. CP保留策略：
   - 防御用CP（甲蔑视等）通常比进攻用CP更有价值
```
