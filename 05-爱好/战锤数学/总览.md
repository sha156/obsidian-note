# 战锤数学（Mathhammer）总览

> 用概率击败对手之前，先用数学击败你的直觉。

---

## 基本概念

**Mathhammer** 是战锤40K（Warhammer 40K）桌游社区中对单位战斗力进行数学分析的方法论。通过概率计算和蒙特卡洛模拟，量化每个单位在不同场景下的伤害输出、耐久能力和性价比，用数据取代"感觉"来指导列表构建和战术决策。

本知识库基于 YouTube 频道 **WarpHammer Math** 的 37 期视频内容整理，频道主播 Blake 是一位数学教师，使用自编程序对每个场景进行 **100万次模拟**，生成完整的概率分布图。

---

## 入门必须知道的10件事

1. **Mathhammer 不是算平均值** — 期望值（均值）具有误导性，完整的分布形态和四分位距（IQR）才能告诉你实战可靠程度
2. **伤害是三道门的乘法** — 命中 × 伤害 × 护甲失败 × 伤害值，每道门独立判定，全部是乘法
3. **Buff 的价值取决于瓶颈** — 当你已经伤害2+时，致命命中几乎无用；当你伤害5+时，致命命中价值翻倍
4. **少量高伤害攻击 = 赌博** — 4发融化枪呈多峰分布，最可能结果可能是"零伤害"
5. **大量低伤害攻击 = 稳定** — 30发爆弹枪呈钟形正态分布，结果高度可预测
6. **无敌护盾（Invul）是可靠性杀手** — 同样4发攻击，打无无敌目标9%弹空，打5++目标23%弹空
7. **Buff 叠加不仅提升伤害，更提升可靠性** — 分布从多峰变钟形，这比伤害数字更有实战价值
8. **每点伤害（Damage per Point）是终极衡量标准** — 240分的单位是军队的1/8，必须多轮存活才能回本
9. **选择 Buff 要根据目标，不是凭直觉** — 以上打下选持续命中，以下犯上选致命命中
10. **75-80%成功率才是可依赖的** — 50%是绝望时的赌博，实战规划要以75%+为底线

## 术语表

| 术语 | 解释 |
|------|------|
| [[期望值/Expected Value]] | 大量重复后的平均结果。D6期望值 = 3.5 |
| [[IQR/四分位距]] | 中间50%结果的范围，衡量可靠性 |
| [[命中门（Hit Gate）]] | 第一关：骰子 >= 命中值即成功。命中3+ = 2/3概率 |
| [[伤害门（Wound Gate）]] | 第二关：力量(S) vs 韧性(T) 决定阈值 |
| [[护甲门（Save Gate）]] | 第三关：AP穿甲修正护甲。AP-2 打 3+ = 实际5+ |
| [[无敌护盾（Invul Save）]] | 不受AP影响的固定保护值 |
| [[无痛（Feel No Pain/FNP）]] | 最终关：受伤后额外骰保，如5+++表示5+无痛 |
| [[持续命中（Sustained Hits）]] | 暴击命中额外算多次命中 |
| [[致命命中（Lethal Hits）]] | 暴击命中自动造成伤害，跳过伤害门 |
| [[毁灭性伤害（Devastating Wounds）]] | 暴击伤害无视护甲保护 |
| [[AP（穿甲值）]] | 修正对手护甲。AP-2 让 3+保护变成 5+ |
| [[重投（Reroll）]] | 允许重新掷骰。重投全部失败 ≈ +50%伤害（命中4+时） |
| [[过热（Hazardous）]] | 射击后掷骰，1=模型受伤。6模型约27%概率损失2+ |
| [[黑暗契约（Dark Pact）]] | 混沌单位特殊规则，以过热风险换取额外能力 |
| [[誓言时刻（Oath of Moment）]] | 星际战士全军机制，选定目标获得重投 |
| [[钓鱼（Fishing）]] | 重投成功值骰子以期获得暴击。大多数时候不值得 |
| [[蒙特卡洛模拟]] | 用计算机运行100万次随机模拟来获得精确分布 |
| [[DPP（每点伤害）]] | Damage Per Point，伤害除以单位点数，衡量效率 |

## 核心数学框架

### 伤害公式（三道门）

```
期望伤害 = 攻击次数 × 命中率 × 伤害率 × 护甲失败率 × 伤害值 × 无痛失败率
```

全部是**乘法**（交换律），计算顺序不影响结果。Blake 的核心技巧：**从右往左（倒序）计算**，极大简化特殊规则处理。

### 力量 vs 韧性对照表

| S vs T 关系 | 伤害阈值 | 概率 |
|---|---|---|
| S >= 2T | 2+ | 83.3% |
| S > T | 3+ | 66.7% |
| S = T | 4+ | 50.0% |
| S < T | 5+ | 33.3% |
| S <= T/2 | 6+ | 16.7% |

## 新手常见误解

| 误解 | 真相 |
|------|------|
| **"算期望值就够了"** | 期望值是平均数，实战中你只打一次——分布形态和IQR才有用 |
| **"Buff 越多越好"** | 叠加有边际递减，某些 Buff 对特定目标价值为0 |
| **"致命命中永远强"** | 当你伤害已经很容易（如2+），致命命中提升仅+1%甚至+0% |
| **"融化枪打什么都强"** | 4发高伤害 = 高方差赌博，打无敌目标弹空率超20% |
| **"大模型最硬"** | 100分重截击者站在目标上对小武器的耐久度可能超过昂贵的精英单位 |
| **"数学决定一切"** | 数学帮你做80%正确决策，但移动、地形、时机同样关键 |
