# 登山杖（越野跑用）

> 登山杖不是"走不动了才用的拐杖"，它是越野跑者的第四肢。

---

## 为什么越野跑要用登山杖？

很多新手觉得用杖是"作弊"或者"体力不行"的表现。事实恰恰相反：

- **UTMB等顶级赛事80%以上选手使用登山杖**
- **部分赛事将登山杖列入强制装备清单**
- 用杖不是因为弱，而是因为聪明

### 核心好处

| 好处 | 说明 |
|------|------|
| **上坡省腿** | 把15-20%的推进力分给手臂，股四头肌和臀大肌负荷显著降低 |
| **下坡稳定** | 多两个支撑点，减少膝盖冲击力30%以上，降低摔倒风险 |
| **维持节奏** | 上坡时用杖建立"手脚交替"节奏，效率比纯走路高 |
| **减少受伤** | 湿滑路段、溪流crossing、夜间视线不好时提供额外平衡 |
| **保护关节** | 长距离下坡中膝盖承受体重3-5倍冲击，杖分担很大一部分 |

### 什么时候用、什么时候收？

| 场景           | 用/收 | 原因       |
| ------------ | --- | -------- |
| 长上坡（>15%坡度）  | ✅ 用 | 借力省腿     |
| 缓上坡/平路       | ❌ 收 | 杖反而成为负担  |
| 技术性下坡（碎石/湿滑） | ✅ 用 | 增加稳定性    |
| 跑得起来的缓下坡     | ❌ 收 | 杖会影响跑步摆臂 |
| 涉溪/过独木桥      | ✅ 用 | 平衡杆作用    |
| 平路快速跑        | ❌ 收 | 拿着累赘     |
| 夜间/大雾（能见度低）  | ✅ 用 | 探路+防摔    |

**关键原则**：杖是工具，不是装饰。该用就用，该收就收。能跑的路段收起来跑更快。

## 核心参数

### 1. 材质

| 材质       | 重量（单根）   | 强度          | 价格          | 适合人群           |
| -------- | -------- | ----------- | ----------- | -------------- |
| **碳纤维**  | 130-180g | 轻但怕侧向力（易折断） | 500-1500元/对 | 竞赛型跑者、追求轻量、不差钱 |
| **铝合金**  | 200-280g | 重但耐操（弯不断）   | 200-600元/对  | 入门跑者、训练用、预算有限  |
| **碳铝混合** | 170-220g | 折中          | 400-800元/对  | 想要轻量又怕碳纤维断的人   |

**核心取舍**：碳纤维轻30-40%但怕**横向冲击**（卡进石缝里一扳就断）；铝合金重但怎么糟蹋都不坏。**竞赛选碳纤，训练/高风险地形选铝合金**。

### 2. 折叠方式

越野跑用的登山杖必须能折叠收纳（塞进背包或挂在背心上），和登山用的伸缩杖有本质区别。

| 折叠方式 | 收纳长度 | 展开速度 | 代表产品 |
|----------|----------|----------|----------|
| **Z折/三折** | 35-40cm | ⚡ 最快（1-2秒甩开） | Black Diamond Distance Z、Salomon NSO |
| **五折** | 25-30cm | 较快（3-5秒） | Leki Micro Trail Pro、Camp Xenon 4 |
| **伸缩锁定** | 50-60cm | 慢（需旋转锁紧） | 传统登山杖（越野跑**不推荐**） |

**越野跑首选 Z 折**：收纳够短挂背包不碍事，甩一下就展开，跑动中1秒完成；五折更短但展开稍慢。伸缩杖太长、太慢，越野跑基本不用。

### 3. 长度选择

越野跑登山杖大多数是**固定长度**（不可调），买的时候就要选对尺寸。

| 身高 | 推荐杖长 |
|------|----------|
| 155-165cm | 105-110cm |
| 165-175cm | 110-120cm |
| 175-185cm | 120-125cm |
| 185cm+ | 125-130cm |

**快速判断**：手握杖柄、杖尖触地时，手肘大约呈**90度或略大**就是合适长度。

**可调长度杖**：部分型号支持±5-10cm调节（如 Leki Micro Trail Vario），好处是上坡缩短、下坡加长；缺点是多了锁定机构，增加重量和故障点。

### 4. 握柄

| 类型 | 特点 | 推荐 |
|------|------|------|
| **EVA 泡沫** | 轻、吸汗、握感舒适 | 越野跑首选 |
| **软木** | 手感好、吸汗、越握越贴手 | 好但偏重 |
| **橡胶** | 耐磨、防滑 | 登山用多，越野跑偏重偏硬 |

越野跑用几乎都是 EVA 泡沫握柄。注意握柄下方是否有**延伸握把**（extended grip），方便在坡度变化时临时降低握持位置。

### 5. 腕带 vs 无腕带

| 方式 | 优点 | 缺点 |
|------|------|------|
| **有腕带** | 发力更高效，杖不脱手 | 摔倒时可能扭伤手腕（杖带着手一起摔） |
| **无腕带** | 摔倒时可以立刻松手保护自己 | 发力时要抓紧杖，手更累 |

**趋势**：越来越多竞赛型选手选择**无腕带**，安全性优先。部分品牌做了可快速脱离的腕带设计（如 Leki Trigger Shark）。

### 6. 杖尖

| 类型 | 特点 |
|------|------|
| **碳化钨尖** | 最耐磨，抓地强，岩石路面首选 |
| **钢制尖** | 便宜耐用，重一些 |
| **橡胶保护套** | 公路/硬地使用时套上，减少噪音和路面磨损 |

越野跑杖尖基本都是碳化钨。有些赛事要求在公路段使用橡胶保护套，带一对备着。

## 使用技巧

### 上坡技术

1. **双杖同步插** — 两根杖同时插在前方，身体推过杖的位置，适合陡坡
2. **交替插杖** — 左手右脚、右手左脚交替，适合中等坡度，维持节奏
3. **杖尖插在脚旁或略后方** — 不要插太前面，会变成"拉"而不是"推"
4. **手臂发力方向是向下推** — 不是拉杖，是用杖把身体"撑"上去
5. **步频与杖频同步** — 上坡走路时建立稳定的手脚节奏

### 下坡技术

1. **杖插在前下方** — 提前放杖建立支撑，身体经过时减速
2. **不要过度依赖杖** — 杖是辅助减震，主要还是靠腿部控制
3. **碎石陡坡用"刹车式"** — 双杖同时插前方，控制下降速度
4. **注意杖尖卡石缝** — 碳纤维杖尤其要注意，卡住硬拔容易折断

### 平路收纳

- Z折杖：折好后**横插在背心包后方弹力带中**，或竖插在背包侧袋
- 很多越野背心包有专门的**杖固定系统**（如 Salomon ADV Skin 系列）
- 收纳要快 — 练到10秒内完成折叠+固定

## 新手选购建议

### 第一对越野跑杖怎么买？

| 问题 | 推荐 |
|------|------|
| 预算有限（200-400元） | **迪卡侬 Evadict 铝合金 Z折杖** — 够用、耐操 |
| 想要兼顾性能和价格（400-700元） | **Black Diamond Distance Z（铝合金版）** — 口碑最好的入门杖 |
| 追求轻量竞赛（700-1500元） | **Black Diamond Distance Carbon Z** — 标杆产品 |
| 背包小想要最短收纳 | **Leki Micro Trail 系列（五折）** — 收纳仅25cm |
| 不确定该买多长 | **BD Distance FLZ 或 Leki Vario** — 可调长度，试错成本低 |

### 买之前确认

- [ ] 确定自己的身高对应杖长（或选可调款）
- [ ] 确认背包有固定杖的位置（多数越野背心包都有）
- [ ] 碳纤维还是铝合金？根据预算和使用场景决定
- [ ] Z折还是五折？看背包空间和展开速度需求

## 保养与常见问题

### 日常保养
- 用完**擦干收好**，泥多的话冲水后晾干
- 检查**弹力绳（Z折内部的连接绳）** 是否有磨损，这是Z折杖最容易坏的部分
- 杖尖磨秃了可以**换杖尖**（各品牌都有配件卖）
- 碳纤维杖检查**杖身有无裂纹** — 有裂纹立刻换，碳纤维不像铝合金会弯，它直接断

### 常见问题

| 问题 | 解决 |
|------|------|
| Z折杖展开不顺畅 | 内部弹力绳可能老化，可更换（DIY或找品牌售后） |
| 杖尖松了 | 用胶水固定或更换杖尖配件 |
| 碳纤维杖折断了 | 无法修复，单根更换（品牌通常卖单根替换杆） |
| 跑步时杖晃动/掉落 | 调整背包固定方式，或使用额外的弹力绳固定 |

## 常见误区

| 误区 | 真相 |
|------|------|
| "用杖是体力不行" | UTMB冠军都用杖，杖是效率工具不是辅助器具 |
| "平路也该用杖" | 平路跑步用杖反而影响效率，该收就收 |
| "越轻越好" | 太轻的碳杖容易断，要根据使用场景权衡 |
| "登山伸缩杖也能用" | 伸缩杖太长太重太慢，越野跑必须用折叠杖 |
| "杖不需要练习就能用" | 上下坡的杖技需要练习，比赛前至少训练3-5次 |
| "一对杖用一辈子" | 弹力绳、杖尖都是消耗品，定期检查更换 |
