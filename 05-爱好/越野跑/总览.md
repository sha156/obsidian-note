# 越野跑 总览

> 离开公路，跑进山野，用双脚感受地球的纹理。

---

## 基本概念

**越野跑（Trail Running）**：在非铺装路面（山地、林道、戈壁、草地等自然地形）进行的跑步运动。与公路跑相比，越野跑更强调地形适应能力、自主导航、装备自给和与自然的互动。核心魅力在于：每一步都不重复，每一场都是冒险。

---

## 入门必须知道的10件事

1. **越野跑不是"在山上跑马拉松"** — 爬升、下降、技术路段让配速毫无参考意义，忘掉公路上的配速概念
2. **走路是越野跑的一部分** — 上坡走、下坡跑是常见策略，专业选手在陡坡也走
3. **一双合适的越野跑鞋是安全底线** — 公路跑鞋的平底在泥石路上会打滑摔倒，越野鞋的抓地齿和防护是刚需
4. **登山杖不是作弊** — 上坡借力、下坡稳定，很多顶级赛事选手都用杖，部分赛事强制携带
5. **学会看等高线和用导航** — 越野跑常在无标记小径，手表GPS或手机离线地图是必备技能
6. **自给自足是基本原则** — 与公路赛不同，越野赛补给点间距长，必须自带水、食物、急救用品
7. **爬升比距离更能衡量难度** — "30公里爬升2000米"比"50公里平路"难得多
8. **下坡比上坡更伤身体** — 下坡的离心收缩对股四头肌破坏极大，需要专门训练
9. **天气变化是最大风险** — 山区天气瞬变，防风防雨外套是强制装备
10. **从短距离开始** — 先跑5-10公里的本地山道，不要一上来就报50公里越野赛

## 术语表

| 术语 | 解释 |
|------|------|
| [[爬升/累计爬升]] | D+，赛道中所有上坡高度之和，如"D+3000m" |
| [[ITRA积分]] | 国际越野跑协会的赛事难度和选手能力评分系统 |
| [[强制装备]] | 赛事要求必须携带的装备清单，不带会被取消资格 |
| [[CP/补给站]] | Check Point，赛道上的补给和检查站 |
| [[关门时间]] | 必须在规定时间内到达CP或终点，否则被强制退赛 |
| [[DNF]] | Did Not Finish，未完赛 |
| [[DNS]] | Did Not Start，报名但未出发 |
| [[FKT]] | Fastest Known Time，某条路线的已知最快完成时间 |
| [[技术路段]] | 需要手脚并用或高度集中注意力的困难地形（碎石、岩壁、陡坡） |
| [[水袋/水囊]] | 装在越野背包内的软体储水装置，通过吸管边跑边喝 |
| [[软水壶]] | 可折叠的软体水瓶，比硬水壶轻且不晃 |
| [[越野背心包]] | 专为越野跑设计的贴身背包，前胸有水壶仓和补给仓 |
| [[落差/Drop]] | 越野跑鞋后跟与前掌的高度差，影响跑姿和脚感 |
| [[齿深/Lug]] | 越野跑鞋底部防滑齿的深度，齿越深抓地越强 |
| [[岩石保护板]] | 鞋底夹层硬板，防止尖石硌脚 |

## 经典赛事（按影响力）

### 世界顶级
1. **UTMB（勃朗峰超级越野赛）** — 环勃朗峰171公里/D+10000米，越野跑界的"奥运会"
2. **Western States 100** — 美国，世界最古老的百英里越野赛
3. **Hardrock 100** — 美国科罗拉多，平均海拔超3000米，极限中的极限
4. **UTMF（环富士山越野赛）** — 日本，环绕富士山约168公里

### 中国知名赛事
1. **UTMB系列赛·中国站** — UTMB积分赛，含多个距离组别
2. **港百（HK100）** — 香港100公里，亚洲最有影响力的百公里越野赛
3. **柴古唐斯越野赛** — 浙江，赛道美、组织好、口碑极佳
4. **环四姑娘山越野赛** — 四川，高海拔、壮美雪山
5. **崇礼168越野赛** — 河北，多距离选择，北方经典赛事

## 新手常见误解

| 误解 | 真相 |
|------|------|
| **"越野跑就是在山上跑步"** | 越野跑融合了导航、装备管理、补给策略、风险判断等多维能力 |
| **"公路跑厉害了再去越野"** | 公路和越野是不同运动，公路慢的人越野可能很强（力量型选手） |
| **"上坡走就是不行"** | 顶级选手在陡坡也走，"power hike"是正式技术 |
| **"越野跑很危险"** | 合理评估能力、带齐装备、量力而行，安全性和徒步登山类似 |
| **"装备越贵越好"** | 适合地形和自己才是好装备，贵的不一定对 |
| **"越野赛就要追求完赛"** | 感觉不对果断退赛，DNF不丢人，安全永远第一 |
